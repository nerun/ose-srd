# Old School Essentials SRD

Este é um **mirror** da tradução não oficial do **Documento de Referência do Sistema (*System Reference Document* — SRD) do *Old School Essentials***, produzida pelo pessoal do [Dados Místicos](https://github.com/pedroleone/ose-srd).

O objetivo é adaptar os arquivos markdown para uma estrutura de site estático mais moderna e acessível com o [mdbook](https://github.com/rust-lang/mdBook), em substituição ao defasado e hermético **Gatsby**.

Este mirror é desenvolvido por um mantenedor autorizado do projeto Dados Místicos e não tem intenção de competir com o projeto original — funcionando, na prática, como um espelho da tradução com uma estrutura alternativa.
